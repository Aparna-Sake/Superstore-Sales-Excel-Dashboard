# 🛒 Superstore Sales Dashboard – Excel Project

## 📌 Project Overview
An end-to-end Excel dashboard project analyzing US Superstore sales data from 2014 to 2017 across 9,994 orders.

## 📊 Key Insights
- Total Revenue: $2,297,201 | Profit: $286,397 | Margin: 12.5%
- Technology has the highest profit margin (17.4%)
- Furniture is loss-risky with only 2.5% margin
- 2017 was the highest revenue year
- West region leads in sales ($725,457)

## 🗂️ Project Structure
- **Dashboard** – Visual KPI dashboard with charts
- **Pivot Tables** – Summary tables by category, region, segment
- **Cleaned Dataset** – 9,994 rows, zero nulls, zero duplicates

## 🛠️ Skills Used
Excel | Data Cleaning | Pivot Tables | Dashboard Design | Data Analysis
